# DASHBOARD-ALMACEN-BICICLETAS
Problema: Una tienda de bicicletas maneja múltiples categorías de productos y niveles de stock, pero carece de una herramienta que consolide las ventas y el inventario en tiempo real para identificar los productos más rentables y los cuellos de botella en el suministro.

Solución: Se desarrolló un dashboard integral que centraliza el rendimiento de ventas y el control de inventarios. El informe permite filtrar por categorías, visualizar indicadores clave de desempeño (KPIs) y optimizar la toma de decisiones basada en la rotación de stock y la demanda del mercado.
